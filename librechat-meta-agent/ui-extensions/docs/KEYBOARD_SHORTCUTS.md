# Keyboard Shortcuts

Mission Control provides comprehensive keyboard shortcuts for efficient navigation and interaction.

---

## Global Shortcuts

These shortcuts work from anywhere in the application.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + N` | New Chat | Start a new conversation |
| `Cmd/Ctrl + K` | Command Palette | Open the command palette |
| `Cmd/Ctrl + /` | Toggle Sidebar | Show/hide the navigation sidebar |
| `Cmd/Ctrl + ,` | Settings | Open settings page |
| `Cmd/Ctrl + B` | Toggle Dark Mode | Switch between light and dark themes |
| `Cmd/Ctrl + Shift + P` | Quick Actions | Open quick action menu |
| `Esc` | Close/Cancel | Close modals, dropdowns, or cancel actions |

## Accessibility Notes

### Screen Reader Support

Mission Control is designed with accessibility in mind:

- All interactive elements have proper ARIA labels
- Keyboard focus is visible and logical
- Screen readers announce state changes
- Images include descriptive alt text

### Focus Management

- `Tab` moves focus forward through interactive elements
- `Shift + Tab` moves focus backward
- Focus is trapped within modals when open
- Focus returns to triggering element when modals close

### Reduced Motion

If you have reduced motion preferences enabled in your operating system, animations will be minimized or disabled.

### High Contrast

Mission Control supports high contrast modes:
1. Use your operating system's high contrast setting
2. Or toggle between Light and Dark modes for best visibility

### Keyboard-Only Navigation

The entire application can be used without a mouse:

1. Use `Tab` to navigate between elements
2. Use `Enter` or `Space` to activate buttons
3. Use `Arrow Keys` to navigate within components
4. Use `Esc` to close dialogs and cancel actions
